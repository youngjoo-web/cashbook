-개인프로젝트 가계부-

1. 회원가입기능 구현
- 아이디중복확인가능
- 사진추가기능
- 비밀번호 확인기능

2. 로그인기능
- 받아온값이 데이터베이스에 일치하면 로그인성공
- 받아온값을 세션값에 저장

3. 가계부 기능
- 가계부데이터 추가
- 추가된 데이터와 멤버아이디에 일치하는 값들을 연산
- 월별합계, 일별합계 기능 구현
- 캘린더형식으로 보이게 기능 구현

4. 게시판 기능
- 게시글 추가는 관리자(member_level 0) 만 가능
- 게시글 수정및 삭제도 관리자만 가능
- 댓글은 관리자 일반회원 모두 가능
- 수정, 삭제는 본인이 쓴 댓글만 삭제 가능
